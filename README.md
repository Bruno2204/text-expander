# Text Expander Extension

Una extensión para navegadores (Chrome/Edge) que permite expandir atajos de texto automáticamente en cualquier campo de texto, optimizando tu tiempo de escritura.

## Características
- **Expansión rápida:** Escribe un atajo corto (por ejemplo, `brb`) y se expandirá instantáneamente a su texto completo (por ejemplo, `Vuelvo enseguida`).
- **Almacenamiento Local:** Los atajos se guardan de forma segura en el almacenamiento local del navegador (`chrome.storage.local`).
- **Fácil configuración:** Incluye una interfaz en el popup para agregar, editar y eliminar atajos fácilmente.

## Instalación en Modo Desarrollador
1. Clona o descarga este repositorio en tu computadora.
2. Abre Chrome o Edge y ve a la página de extensiones (`chrome://extensions` o `edge://extensions`).
3. Activa el **Modo de desarrollador** (Developer mode) en la esquina superior derecha.
4. Haz clic en el botón **Cargar descomprimida** (Load unpacked).
5. Selecciona la carpeta donde guardaste este proyecto.
6. ¡Listo! La extensión ya estará activa.

## Uso
Simplemente haz clic en el ícono de la extensión para abrir el menú emergente y comienza a agregar tus propios atajos de texto. Luego, ve a cualquier página web, escribe tu atajo y el texto se expandirá automáticamente.
